## Backend

### Setup

```bash
npm init -y
```

## Frontend

### Setup

```bash
npm create vite@latest
```

### Tailwind CSS

#### Installation

```bash
npm install tailwindcss @tailwindcss/vite
```

#### Config (in vite.config.ts)

```bash
import tailwindcss from '@tailwindcss/vite'
```

```bash
plugins: [
    tailwindcss(),
],
```

#### Import in index.css

```bash
@import "tailwindcss";
```
